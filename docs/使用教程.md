> 使用方式：打开终端，把下面的命令直接**复制粘贴**运行即可。
> 每行 `#` 开头的是注释说明，不会被执行。

## ▶ API Key 配置

```bash
# 设置 DeepSeek 的 Key（推荐首选，性价比最高）
video-agent key set --provider deepseek

# 设置 Kimi 的 Key（长文本能力强）
video-agent key set --provider kimi

# 设置豆包的 Key（字节跳动，国内访问稳定）
video-agent key set --provider doubao

# 设置通义千问的 Key（阿里云，中文理解好）
video-agent key set --provider qwen

# 设置智谱 GLM 的 Key（有免费额度）
video-agent key set --provider zhipu

# 查看所有 Key 的配置状态（显示哪些已配置、哪些未配置）
video-agent key list
```

> 输入 Key 时终端不显示任何字符，这是正常的安全行为，输完直接回车即可。

## ▶ 核心功能：爆款文案复刻（完整四步流程）

```bash
# 启动完整创作流程（菜单引导，按提示操作）
video-agent
# 然后选「🚀  开始创作（完整四步流程）」
```

**流程说明：**

```
Step 1  输入爆款文案
        ├── 方式A：直接粘贴 → 粘贴完在新一行输入 ===END=== 结束
        ├── 方式B：输入文件路径（如 /Users/你的名字/Desktop/文案.txt）
        └── 方式C：从知识库里选已存入的文案

Step 2  AI 分析爆款框架（自动完成，等待即可）
        └── 分析完成后，直接回车使用全部结果；或选"否"手动筛选

Step 3  输入新选题
        ├── 填写选题（如：高三最后30天数学冲刺攻略）
        ├── 填写目标字数（默认1000字）
        └── 填写额外要求（可留空，如：开头用反问句、结尾必须推课）

Step 4  导出 Word 文档
        └── 确认保存路径（默认桌面），生成 .docx 文件
```

生成的 Word 文档包含：**正式稿**（可直接录制）、**注释版**（标注结构）、**创作说明**、**分析报告附录**。

## ▶ 快速创作（无需爆款参考，直接写）

```bash
# 最简用法：输入选题直接生成（字数默认1000字，保存到桌面）
video-agent create --topic "高三冲刺100天，数学从90分到140分"

# 指定字数
video-agent create --topic "英语阅读理解的核心误区" --words 1200

# 指定保存目录
video-agent create --topic "化学备考技巧" --words 800 --output ~/Desktop

# 把选题换成你自己的，其他参数照抄即可
video-agent create --topic "【改这里写你的选题】" --words 1000 --output ~/Desktop
```

## ▶ 快速分析（只分析爆款，不创作）

```bash
# 分析单篇文案文件
video-agent analyze /Users/yangheyao/Desktop/爆款文案.txt

# 分析多篇（AI 会综合提炼共同框架）
video-agent analyze /Users/yangheyao/Desktop/文案A.txt /Users/yangheyao/Desktop/文案B.txt

# 分析并把报告导出为 Word 文件到桌面
video-agent analyze /Users/yangheyao/Desktop/爆款文案.txt --output ~/Desktop
```

> 文件路径：把文案保存为 `.txt` 文件，右键 → 显示简介，复制路径粘贴进来。

## ▶ 自由对话（和 AI 随意聊，不限场景）

```bash
# 启动自由对话

```

进入对话后可以问任何问题，例如：
- "帮我想5个关于高考数学的爆款选题"
- "这段话改得口语化一点：[粘贴你的文字]"
- "开头钩子怎么写更吸引人？"

**对话中的特殊命令：**

```

/exit     → 退出对话，回到命令行
Ctrl + C  → 强制退出
```

## ▶ 知识库管理

知识库的作用：存放优质文案，AI 创作时自动参考你的风格。

```bash
# 查看知识库里的所有文章（会显示 ID、标题、标签等）
video-agent kb list

# 添加一篇文案到知识库（最简用法）
video-agent kb add /Users/yangheyao/Desktop/好文案.txt

# 添加时同时设置标题、标签、平台
video-agent kb add /Users/yangheyao/Desktop/好文案.txt --title "数学冲刺爆款" --tags "数学,高考" --platform "B站"

# 删除知识库里的文章（先用 kb list 查看文章ID，再填入下面命令）
video-agent kb remove 【粘贴文章ID，如 abc123】

# 进入知识库的交互式管理菜单（可视化操作）
video-agent
# 然后选「📚  知识库管理」
```

## ▶ 记忆与偏好设置

记忆设置让 AI 每次都了解你是谁、你的风格是什么，**一次设置永久生效**。

```bash
# 进入记忆管理菜单（可视化操作）
video-agent
# 然后选「🧠  查看/编辑记忆」
```

**菜单里可以设置的内容：**

| 功能 | 作用 |
|------|------|
| 查看当前记忆 | 看看 AI 目前记住了什么 |
| 添加创作要求 | 告诉 AI 你的固定要求，每次创作自动执行 |
| 删除创作要求 | 删除某条不再需要的要求 |
| 编辑偏好 | 设置创作领域、主要平台、人设描述、文风标签、默认字数 |
| 清空所有记忆 | 删除全部记忆，重新开始（有确认提示） |

**推荐填写的自定义创作要求示例（进菜单逐条添加）：**

```
不要在开头用"大家好"
每段不超过4句话
结尾必须有引导关注或点击主页的话术
数据必须具体，不要说"很多人"，要说"73%的学生"
不要用感叹号超过3个
开头第一句必须是钩子，制造悬念或反差
```

## ▶ 常见问题速查

**提示"未找到 API Key"**

```bash
# 重新设置（把 deepseek 换成你用的提供商）
video-agent key set --provider deepseek

# 确认 Key 是否已保存
video-agent key list
```

**命令找不到 / 提示 command not found**

```bash
# 重新激活环境
cd /Users/yangheyao/Desktop/workflow/video-agent
source .venv/bin/activate

# 如果还不行，重新安装
.venv/bin/pip install -e .
```

**生成的文案字数不够**

```bash
# 在创作时的"额外要求"里加上这句话：
# 严格要求正式稿不少于1000字，不允许缩水
```

**想换模型重新生成**

```bash
# 切换模型后重新运行创作命令即可
video-agent model
```

**Word 文档找不到了**

默认保存在桌面，文件名格式为 `选题名称_年月日时分.docx`。
可在「⚙️ 模型与配置」→「📁 修改默认输出目录」里改为其他位置。
